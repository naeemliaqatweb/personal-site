PLD-Literacy.org

I worked on PLD-Literacy.org, a WooCommerce-based platform specializing in selling PDF products. The project involved extensive customization and optimization.

  • Created a custom Gutenberg block for YouTube video popups, allowing dynamic video ID input and popup display functionality.
  • Implemented custom license key management, including activation, deactivation, and transfer logic.
  • Displayed digital products to users based on license and access status.
  • Developed a custom post type and set up a scheduled task (every 15 minutes) to:
    • Check a directory for new video uploads
    • Automatically fetch the video name
    • Create a new post with the video and assign a featured image
  • Completed and customized the full WooCommerce user flow based on Figma designs, including:
    • Product page
    • Search page
    • Cart page
    • Checkout page
    • My Account page
    • Add Payment page
    • User Orders page
    • User Subscriptions page
    • User Profile page
  • Integrated Emailable API to validate user email addresses before registration, allowing only valid emails to register.
  • Implemented performance optimizations including:
    • Object cache
    • WP Rocket for Cloudflare cache, browser cache, and minification of CSS/JS
    • Imagify for image compression (WebP format)
    • Lazy loading for images to improve site speed
  • WooCommerce Page Customization: Designed and customized search, product, cart, checkout, register, and login pages using Figma designs.
  • Template & Page Customization: Enhanced and redesigned multiple pages to align with the client's branding.
  • WooCommerce Email Customization: Created custom email templates for all WooCommerce email notifications.
  • Post Page Redesign: Revamped post pages for a better content presentation.
  • Subscription & Invite Subscription Customization: Implemented custom subscription functionality with invite options.
  • Key Plugins Used: WooCommerce, Postmark, Dear PDF, WP Rocket, Object Cache, Layer Slider, Gravity Forms, Drip, Advanced Custom Fields PRO, Ajax Search Pro, AutomateWoo, Crisp for WooCommerce, Imagify, reCAPTCHA for WooCommerce, UpdraftPlus, WaterWoo PDF Premium, WC Serial Numbers, WooCommerce Eway Payment Gateway, WooCommerce Xero Integration, Wordfence Security, YITH WooCommerce Wishlist, Yoast SEO.
