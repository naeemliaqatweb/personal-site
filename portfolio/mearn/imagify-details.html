Description:
Imagify is an AI-powered web application that generates images from text prompts using the ClipDrop API. Built with the MERN stack (MongoDB, Express.js, React, Node.js), it allows users to create realistic AI-generated images easily and intuitively.

ðŸŒŸ Key Features:

  • User Authentication: Secure login and registration system with password hashing.
  • Free Credit System: Every new user is automatically given 5 free credits to generate up to 5 images for free.
  • AI Image Generation: Users enter a text prompt, and an image is generated using the ClipDrop Text-to-Image API. Each generation deducts 1 credit.
  • Subscription Plans: After using free credits, users can purchase one of the following plans to continue:
    • Basic
    • Advanced
    • Pro
  • Downloadable Images: Generated images are displayed on the user dashboard with a download option.
  • Admin & User Roles (optional): Role-based access control for managing users and plans.
