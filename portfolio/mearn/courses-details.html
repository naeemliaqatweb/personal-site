Description:
EduMaster is a dynamic online course web application built using the MERN stack (MongoDB, Express.js, React, Node.js). It offers a seamless learning experience through interactive course pages, a secure authentication system, and a fully functional admin dashboard for managing courses, users, and enrollments.

ðŸŒŸ Key Features:

  • Home Page: Modern, responsive landing page featuring top-rated courses, testimonials, CTAs, and instructor highlights.
  • Courses Page: Browse and search all available courses with filters (category, level, price) and pagination.
  • Course Details: Each course has a detailed page with syllabus, video preview, instructor info, and enrollment option.
  • Pricing Page: Lists Free and Premium plans with features comparison. Paid plans unlock full course access and certifications.
  • User Authentication: JWT token-based login and registration system with secure password hashing.
  • User Dashboard: Track enrolled courses, progress, certificates, and edit profile info.
  • Admin Panel: Admins can manage users, create/edit/delete courses, upload content, and monitor enrollments.
  • Role-Based Access: User and Admin roles with protected routes and middleware authorization.
  • Session Management: Secure JWT token stored in HTTP-only cookies for authentication sessions.
